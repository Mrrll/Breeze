## Instalación
###### Instalación de Breeze


>Typee: en la Consola:
```console
composer require laravel/breeze --dev
```

>Typee: en la Consola:
```console
php artisan breeze:install

php artisan migrate
npm install
npm run dev
```

>Typee: en la Consola:
```console
php artisan serve
```
